# HooMee Coffee

Personal static website for `www.hoomee.cc`.

## Structure

- `index.html` - homepage
- `info.html` - tech and information posts
- `life.html` - lifestyle posts
- `pet.html` - BanBan pet gallery
- `about.html` - about page
- `article-ai-weekly.html` - long-form article page
- `css/style.css` and `js/main.js` - shared styles and behavior for shared-template pages

## Deployment

This repository is published with GitHub Pages from the repository root.

- `CNAME` points to `www.hoomee.cc`
- `.nojekyll` keeps GitHub Pages from applying Jekyll processing
- `404.html` provides a simple fallback page
